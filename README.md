- **季节性促销码**：黑五、新年等节点，官方会在公告页或客服工单系统里发布，折扣幅度通常是循环折扣（每月续费都打折），不是一次性减免

其实更准确的说法是：DMIT 的"折扣"更多体现在**选对付款周期**，而不是靠一个神奇的优惠码。

## DMIT 套餐线路怎么选，CN2 GIA 和软银有什么区别

很多人买 DMIT 之前搞不清楚线路差异，结果买了不适合自己的套餐。这里拆开说。

### 三条主力线路的实际体验

**CN2 GIA（Premium）**
这是 DMIT 的招牌线路，去程和回程都走电信 CN2 GIA，理论上是目前商用线路里对中国大陆最友好的。我自己用下来，北京、上海、广州三个方向的延迟都比较稳，晚高峰波动不大。适合对延迟敏感的场景，比如需要实时连接国内服务器的业务。

**软银（Lite / Eyeball）**
回程走日本软银，对联通和移动用户更友好，电信用户体验相对弱一些。价格比 CN2 GIA 低，适合预算有限但主要用联通宽带的用户。

**普通优化线路**
部分入门套餐走的是普通 BGP 或优化线路，价格最低，但高峰期稳定性不如前两者。适合对线路要求不高、主要用来建站或跑脚本的场景。
